# 统一聚合平台SDK

### 一、项目介绍
* 此项目分别对GroMore、TopOn两家聚合平台SDK进行统一api标准二次封装。两个SDK分别适配穿山甲、快手、优量汇等平台广告，其它平台广告请自行添加SDK进行自测！
### 二、功能特性
* 开屏广告(提供超简单开屏广告展示能力)
* 激励视频广告(提供超简单激励视频播放能力)
* 插屏广告(提供超简单插屏广告展示能力)
* 信息流广告(提供超简单信息流展示能力)
* banner广告(提供超简单banner展示能力)
